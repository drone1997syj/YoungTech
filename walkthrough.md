# 영테크 (YoungTech) 쇼핑몰 인도 가이드

영테크 산업용 서보모터 및 서보드라이버 온라인 쇼핑몰이 성공적으로 구축되었습니다. 본 문서는 시스템 아키텍처, 기능 명세 및 테스트 시나리오를 안내합니다.

---

## 1. 기술 스택 및 아키텍처

- **Core**: React 18, Vite 6
- **Styling**: Vanilla CSS (CSS Variables 기반 디자인 시스템 구축)
- **State & Routing**: `React Context API` (`ShopContext.jsx`) 기반의 단일 페이지 상태 제어 네비게이션 구조
- **Mock Database**: `LocalStorage` 연동 Mock DB 서비스 (`mockDb.js`) 구축 (주문, 회원 정보, 장바구니, 상담 이력 관리)
- **Icons**: `lucide-react`

---

## 2. 주요 구현 기능 및 디자인 특징

### 💜 보라색 시그니처 테마 및 공식 브랜드 이미지 적용
- `src/index.css`에 핵심 브랜드 컬러(`--primary: #7c3aed`, `--secondary: #a78bfa`, `--accent`, `--dark` 등)를 정의하여 일관된 고급 하이테크 레이아웃 유지.
- 헤더 영역에 **영테크 공식 로고(`https://globalyt.co.kr/sh_img/hd/top_menu/logo.png`)**를 직접 활용하고, 실제 쇼핑몰 취급 품목인 **모터, 감속기, 로봇, PLC, 볼스크류/LM** 카테고리로 전면 개편.
- 홈 화면 최하단에 유명 글로벌 FA 메이커(PANASONIC, MITSUBISHI, SHIMPO, MOONS', INOVANCE 등)의 **FA BRAND PARTNERS** 협력 띠 영역을 추가하여 신뢰감 있는 브랜드 유통망 시각화.

### 🤖 실시간 가상 챗봇 위젯 (`ChatWidget`)
- 우측 하단 고정 플로팅 버튼을 통해 활성화.
- **공식 품목 맞춤형 키워드 인지 자동응답**: `모터`, `감속기`, `로봇`, `PLC`, `도면`, `카달로그` 등의 실무 키워드에 최적화된 상세 제품 상담 및 자료실 가상 링크 안내 기능 추가.

### 📊 동적 사양 비교 모듈 (`ProductCompare`)
- 상품 리스트나 상세 페이지에서 `비교` 버튼 클릭 시 상단에 접이식 사양 비교창 노출 (최대 3개).
- 비교 품목의 모든 specs 키 값을 추출하여 유니크 사양(메이커, 분해능, 최대 속도, 제어 방식 등)을 행(Row)으로 나열하는 고품질 명세 테이블 레이아웃.

### 💳 가상 신용카드 PG 결제 모듈 (`MockPGModal` / `Checkout`)
- KG이니시스/토스페이먼츠 형태를 모방한 독립 결제 창.
- **모의 시나리오 선택 가능**: 결제 성공, 한도 초과 오류, 카드 번호 에러 등 테스트할 결과를 선택하여 실제 승인 요청 대기 흐름(3초 스피너 로더)을 가상으로 재현.

### 📦 마이페이지 및 가상 배송추적 (`MyPage`)
- 결제 완료 시 마이페이지 주문 내역에 즉시 추가.
- 주문 내역 상세 영수증 정보 확인 및 **가상 배송 타임라인**("결제 완료" -> "배송 준비중" -> "배송중" -> "배송 완료") 트래킹 컴포넌트 탑재.

## 3. 원클릭 테스트 및 검증 가이드

1. **가상 로그인 테스트**
   - 상단 헤더 우측 자물쇠 아이콘 클릭 -> 로그인 페이지 이동.
   - [계정 자동입력] 버튼 클릭 시 테스트 계정 ID: `test` / PW: `password` 자동 완성 후 가상 로그인 완료.
2. **장바구니 및 사양 비교**
   - 제품 목록에서 원하는 모터/드라이버의 `비교`를 눌러 스펙 테이블을 비교해 보고 `장바구니 담기` 클릭.
3. **가상 결제 흐름 검증**
   - 장바구니 페이지에서 `주문서 작성하기` 선택.
   - 배송 수령 정보를 확인한 후 `가상 결제창 열기` 클릭.
   - 카드 번호 입력 및 시나리오(예: 결제 성공)를 선택하고 `결제 승인 요청` 실행 -> 3초 후 승인 완료 메시지와 함께 영수증 자동 생성.
   - 마이페이지로 이동하여 배송 타임라인 및 상세 영수증 확인.
